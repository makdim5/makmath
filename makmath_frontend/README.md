# MakMath Frontend

This project is the frontend part of the MakMath application, built using Vite.

## Running the Project

1. Install dependencies:
   ```
   npm install
   ```

2. Start the project in development mode:
   ```
   npm run dev
   ```

The project will be available at the address shown in the console (usually http://localhost:5173).

## Technologies Used

- [Vite](https://vite.dev/guide/) - a fast build tool and development server for modern web applications.
